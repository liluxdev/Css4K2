Css4K2
======

This K2 plugin enables 1 css file for 1 k2 template.

If you've created a k2 template called 'mytpl' and affected it to a k2 category, the plugin will automatically add the css file 'templates/YOURJOOMLATEMPLATE/html/com_k2/templates/mytpl/mytpl_style.css' to the document stylesheet list when the k2 category or an item of this category will be displayed.

Compatibility with Joomla! versions

v1.2.x : J!1.5
v1.7.x : J!2.5
v1.8.x : J!2.5 & J!3.0

You can find & request for translations here : https://www.transifex.com/projects/p/Css4K2/
